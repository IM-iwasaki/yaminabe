コード規約

・命名規則
クラス名　	PascalCase
変数名		camelCase
定数		ALL_CAPS
関数名		PascalCase
ファイル名	クラス名と一致
enumや構造体	定義とメンバともにPascalCase


・コードフォーマット
インデント	タブ1つ

・｛｝の位置
if (condition) {
    ...
}
else {
    ...
}

・コメント
クラスの最上部にどんなクラスかコメントを明記する
各関数にも同様にコメントを付ける


・関数・クラス設計
１クラスは１役割を持つようにする

・クラス順序
変数→イベント関数→自作関数


・禁止事項
理由のないマジックナンバー使用禁止
Findの多用禁止
Debug.Logの多用禁止
自分以外のPrefab・ScriptableObjectを勝手に上書きしない
